# Страница для администрирования учётных записей
### _Тестовое задание при приёме на работу_

#### Технологии:
- Vue.js
- TailwindCSS
- TypeScript
- Pinia
- Zod
- Vite

## Настройка
После клонирования репозитория, установите зависимости:
```
npm i
```
Для запуска локального сервера (http://localhost:5173/):
```
npm run dev
```
